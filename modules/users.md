# Users

## Objectif

Definir le module utilisateurs d'ATLAS.

## Regles officielles

Un utilisateur est un compte rattache a une organisation.

Chaque compte doit avoir un role clair.

## Fonctionnement

Le module gere les comptes, les invitations, les roles, les droits, le statut du compte et l'historique.

## Dependances

Organisations, roles, securite et notifications.

## Permissions

La gestion des comptes est reservee aux administrateurs autorises.

## Automatisations

Invitation et rappels peuvent etre automatises sous controle.

## Points de vigilance

Eviter les droits trop larges par defaut.

## A confirmer

- Parcours d'invitation.
- Roles initiaux.
